Slotter

Python FullStack App
TechStack:

React, JavaScript, Express.JS, MongoDB, CSS.

GitHub here

The Slotter app is a slot machine full-stack app. The app follows the same functionality of a 1980s fruit-based slot machine.

It has CRUD functionality for adding cash, displaying the user's current balance, ability to hold any of the wheels before the next spin, selecting different users, and cashing out.

The app also has images for icons drawn from an API and a visual display of win and loss streaks, with an admin feature allowing the adjustment of play options, including cost per play and value per win. The admin can adjust play options such as cost per play and value per win.

The app uses a non-relational DB as it is well-suited for handling large volumes of unstructured data. The backend was built in Express.js due to its ease of use and simplicity. It offers a straightforward way of handling HTTP requests and responses. One of the strengths of this app is that it allows for efficient development, good performance, and easy scalability.
