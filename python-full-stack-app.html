Allerg.ly

Python FullStack App
TechStack:

Flask, SQL, Python, HTML, CSS.

View here
GitHub here

Allerg.ly is a full-stack web application that allows users to create, read, update, and delete (CRUD) products.

The app is built using HTML and CSS for the frontend, with Python and Flask as the backend web framework. Jinja2 is used as the templating engine to render the dynamic web pages based on user requests.

The app uses a PostgreSQL database to store product information and leverages RESTful routes for API design, which allows the frontend to communicate with the backend using HTTP methods such as GET, POST, PUT, and DELETE. This enables the app to provide users with efficient and real-time updates to the product catalog, enhancing the overall user experience.

One of the strengths of this app is its adherence to object-oriented programming (OOP) and model-view-controller (MVC) design principles. This allows for a clear separation of concerns between the frontend, backend, and database layers of the app, making it easier to maintain and scale as the user base grows.
